# Malooley Insurance Website

Website for **Malooley Insurance** (Michele A. Malooley), an independent Medicare and insurance advisor based in Boynton Beach, Florida.

This is a rebuilt, editable static version of the site previously hosted on Wix, so changes can be made directly through GitHub and hosted via GitHub Pages.

## Structure

- `index.html` &mdash; Home page (hero, specialties, why choose Michele, bio/credentials, testimonials preview + Google Reviews widget, service areas, contact)
- `testimonials.html` &mdash; Full testimonials page + Google Reviews widget
- `resources.html` &mdash; Medicare & Social Security resource links
- `blog.html` &mdash; Blog posts
- `style.css` &mdash; All site styling (colors, layout, fonts)
- `script.js` &mdash; Mobile nav toggle + footer year

## To Do / Next Steps

1. **Add real photos**: Upload Michele's headshot and any background images to this repo (e.g. as `photo.jpg`) and reference them in `index.html` in place of the placeholder hero-photo box.
2. **Add the Google Reviews widget**: Each page has a clearly marked "Google Reviews widget goes here" box in `index.html` and `testimonials.html`. Replace it with your chosen embed (Google Places API, Elfsight, EmbedSocial, etc.) and your Business Place ID.
3. **Connect the contact form**: The contact form on the home page is currently a placeholder. Connect it to a form backend like Formspree, Getform, or a Google Form so submissions reach your inbox.
4. **Enable GitHub Pages**: In this repo's Settings > Pages, set the source to the `main` branch to get a free live preview URL.
5. **Point your domain**: Once you're happy with the site, update your GoDaddy DNS settings for malooleyinsurance.com to point to GitHub Pages (or your chosen host).
6. **Access for the business owner**: Add Michele as a Collaborator (Settings > Collaborators) so she always has access to this repository independent of who manages day-to-day updates.

## Editing
